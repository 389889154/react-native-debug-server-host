
# Debug Server Host For iOS

## React Native服务器地址更换

运行时更换服务器地址，不用为更换调试服务器地址而频繁打包。android调试状态有更换服务器地址功能，而iOS不具备此功能，本库就是为iOS提供此功能。

![](./image/1.png)

## 安装

直接将DebugServerHost文件夹添加到项目中即可，无需任何设置。

## 使用

推荐将url生成二维码，使用二维码扫描代替手工输入，点击OK时自动reload新服务器地址页面。

![](./image/2.png)
